# Intelligent-Spelling-Grammar-Analyzer
LexoCheck is a simple text analysis tool built using compiler design concepts. It processes input text through stages like tokenization, lexical analysis, and rule-based parsing to detect spelling and basic grammar errors. The system checks each word using a dictionary, suggests corrections, and identifies common issues such as repeated words, missing spaces, and incorrect capitalization.

This project demonstrates how core compiler principles can be applied to natural language processing to create a structured and explainable spell-checking system.
